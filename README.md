# Light Clicker
An extremely lightweight and portable Auto-Clicker with zero dependencies, installers or config files.

This was intended for personal use-cases so it's missing a lot of extra functionality.

## Usage

Download *lightclicker.exe* from the [releases](https://github.com/JeremyJenki/Light-Clicker/releases) page. Place the file anywhere you want and run. To activate the autoclicker, hold down the hotkey.

You can use the tray icon to select between three speed options: Normal = 10 cps, High = 20 cps, and Max = Uncapped.

### Hotkey
`Hold Left Alt` + `LMB`
