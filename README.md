# Light Clicker
An extremely lightweight and portable Auto-Clicker with zero dependencies, installers or config files.

This was intended for personal use-cases so it's missing a lot of extended functionality like random offsets or customisable hotkeys.

### Hotkey:
`Hold Left Alt` + `LMB`

## Speeds
 
| Mode   | Interval | CPS      |
|--------|----------|----------|
| Normal | 100ms    | 10       |
| High   | 50ms     | 20       |
| Max    | 1ms      | 1000     |
